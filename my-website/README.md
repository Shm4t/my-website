# HIT — Современный лендинг с видеовставкой и Bootstrap каруселью

Адаптивный одностраничный сайт с видеовставкой на фоне, каруселью изображений и интерактивными элементами. Стилизован под современный дизайн с градиентными фонами и анимациями.

## 🚀 Демо

Сайт доступен по адресу:  
`https://[ваш-username].github.io/[название-репозитория]/`

## 📋 Особенности

- **Адаптивный дизайн** — корректно отображается на всех устройствах (десктоп, планшет, мобильные)
- **Видео на всю ширину** — автоматически воспроизводится в фоне с затемнением
- **Bootstrap 5 карусель** — слайдер с изображениями и текстовым описанием (работает на всех устройствах)
- **Боковая панель** — информационные блоки с ссылками "Learn More"
- **Социальные иконки** — стилизованные кнопки с эффектом наведения (Spotify, Pinterest, Dribbble, Telegram)
- **Анимированный Telegram логотип** — появляется с эффектом "въезда"
- **Градиентный фон** — радиальный градиент от синего к белому

## 🛠️ Технологии

- **HTML5** — семантическая разметка
- **CSS3** — кастомные стили, анимации, градиенты, медиазапросы
- **Bootstrap 5.2.3** — сетка, компоненты карусели, адаптивность
- **JavaScript (Bootstrap JS)** — для работы карусели и интерактивных элементов
